Bootstrap

Bootstrap es uno de los actual framework mas populares del momento.

Características Principales

Bootstrap es un **framework de componentes (Component-First)** que utiliza HTML5, CSS3, y se apoya en jQuery/Javascript para sus elementos interactivos. Su núcleo es un sistema de **Grid (rejilla) de 12 columnas** que simplifica el diseño responsive. Ofrece una vasta colección de componentes UI predefinidos como modales, carruseles, y barras de navegación.

Ventajas

  • Desarrollo Rápido: Ofrece un conjunto de componentes predefinidos que aceleran el desarrollo web.
  • Diseño Responsive: Facilita la creación de sitios web adaptables sin necesidad de escribir muchas *media queries*.
  • Gran Comunidad y Documentación: Cuenta con una comunidad activa y documentación detallada que proporciona un soporte constante.
  • Coherencia de Diseño: Sus estilos predefinidos ayudan a mantener una imagen de marca uniforme.

Inconvenientes

  • Tamaño del Archivo: Sus archivos CSS y JS pueden ser grandes, lo que puede aumentar el tiempo de carga si no se optimiza (cargando código no utilizado).
  • Diseños Similares: Si no se personaliza, el diseño puede parecer genérico, similar a muchos otros sitios web que usan el tema predeterminado.
  • Código HTML Complejo: A veces requiere modificar el HTML con múltiples contenedores para que los componentes funcionen correctamente.

Componentes

  • Sistema de Rejilla (Grid System): Es la funcionalidad más fundamental. Utiliza el modelo de 12 columnas y las clases container, row, y col- para crear diseños responsivos y estructurados.
  • Navbar: Un componente complejo y listo para usar que incluye branding, enlaces de navegación, formularios de búsqueda y soporte para dropdowns (menús desplegables), con colapso automático en dispositivos móviles (requiere JS).
  • Carrusel: Un componente de diapositivas que permite alternar contenido (imágenes o texto) dentro de un ciclo predefinido. Es el ejemplo perfecto de un componente que requiere la inclusión de Javascript para su funcionamiento interactivo.

Instalacion

Bootstrap requiere tanto el archivo CSS en el <head> como el Javascript al final del <body> para los componentes interactivos.

Estructura HTML para la inclusión:

<!-- CSS de Bootstrap en el head -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JS de Bootstrap (al final del body) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
